# IronClaw Bounty Task Analyzer

## What Problem It Solves
Web3 contributors often receive bounty task descriptions that are vague or complex. 
This skill analyzes any bounty task and instantly breaks it down into a clear action 
plan, so contributors know exactly what to build and submit — reducing rejections.

## How It Works
1. User pastes a bounty task description into the IronClaw agent
2. The skill reads the description and identifies key requirements
3. It returns a structured breakdown with:
   - A checklist of exact requirements
   - List of deliverables needed for submission
   - Step-by-step action plan
   - Common rejection reasons to avoid

## Use Case
Ideal for NEAR Legion contributors, IronClaw bounty hunters, and any Web3 
community member navigating bounty campaigns.
